# AI Crash Course for Absolute Beginners

A complete, self-contained 25-lesson course built with pure HTML/CSS — no frameworks, no build tools, no account required. Designed for complete beginners who want to understand modern AI from the ground up.

**Instructor:** Abena Fosuaa Gyasi

---

## Course Structure

| Phase | Topic | Lessons |
|-------|-------|---------|
| 1 | What is AI? | 1.1 – 1.4 |
| 2 | Data & Mathematics | 2.1 – 2.5 |
| 3 | Machine Learning | 3.1 – 3.6 |
| 4 | Deep Learning | 4.1 – 4.5 |
| 5 | AI in the Real World | 5.1 – 5.5 |

## Practice Notebooks

15 hands-on Google Colab notebooks covering every code-bearing lesson. See [notebooks/README.md](notebooks/README.md) for the full list with direct Colab links.

## Quick Start

Open `ai_crash_course_pro.html` in any browser — no server required.
