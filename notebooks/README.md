## How to Use

1. Click any **Open in Colab** badge above
2. Sign in with your Google account (free)
3. Click **Runtime > Run all** to execute every cell
4. Edit and experiment — you cannot break anything!

## Requirements

All notebooks install their dependencies automatically in the first cell. A free Colab GPU is recommended for Lessons 4.3 and above.
